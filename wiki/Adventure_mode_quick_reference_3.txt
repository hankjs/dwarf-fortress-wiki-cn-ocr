These are the [[controls]] used in [[Adventurer mode]] (''or perhaps you were looking for a [[Adventure mode quick start|quick start]], instead?'').
==Key Bindings==

{{KReference|Movement|
*{{k|8|2|4|6|7|9|1|3|↑|↓|←|→}} - Move
*{{k|Alt}}+(movement key) - Move carefully or enter dangerous terrain
*{{k|Shift}}+(movement key) - Move in direction and upwards
*{{k|Ctrl}}+(movement key) - Move in direction and downwards
*[[File:Adv ui ascend.png]]{{k|<}} or {{k-|Shift|5}} (num lock off) - Ascend
*[[File:Adv ui descend.png]]{{k|>}} or {{k-|Ctrl|5}} (num lock off) - Descend
*{{k|.}} - Wait for 10 instants
*{{k|,}} - Wait for 1 instant
*{{Adv menu icon|j}} - Jump. Sprint in movement options to run and jump.
*{{Adv menu icon|h}} - Start climbing by grabbing {{k|h}}old of a wall or tree; mount and lead pets
*{{Adv menu icon|s}} - Stand or lie down*
*{{Adv menu icon|S}} - Sneak
*{{Adv menu icon|T}} - Fast Travel
*{{Adv menu icon|m}} - Open movement preferences interface
}}
<nowiki>*</nowiki>Use {{k|s}} to lay down, and then crawl past NPCs blocking your way.

{{KReference|Combat|
*{{k|8|2|4|6|7|9|1|3|↑|↓|←|→}} - Random attack on adjacent creature
*{{Adv menu icon|A}} - Attack an adjacent creature
*{{Adv menu icon|W}} - Wrestle an adjacent creature
*{{Adv menu icon|f}} - Fire a projectile
*{{k|t}} - Throw an item
*{{Adv menu icon|X}} - Use ability (pet an animal, spit at someone...)
*{{Adv menu icon|C}} - Open combat preferences interface
}}

{{KReference|Environment|
*{{k|l}} - Look around
*{{Adv menu icon|L}} - Search the nearby area very carefully (chance of discovering [[Vermin|small creatures]])
*{{Adv menu icon|k}} - Talk to somebody
*{{Adv menu icon|P}} - Perform or compose
*{{Adv menu icon|K}} - Display tracks
*{{k-|Alt|k}} - Describe tracks underfoot.
*{{Adv menu icon|c}} - View companion interface
*{{Adv menu icon|E}} - Party tactical settings.
*{{k|o}} - Describe [[Smell|odor]].
*{{k|O}} - Show strongest odor.
*{{k|u}} - Interact with lever, building, furniture, or mechanism.
*{{Adv menu icon|y}} - Assume or abandon a submissive position (yield).
}}
| valign='top' |
{{KReference|Items|
*{{Adv menu icon|i}} - Inventory
*{{Adv menu icon|d}} - Drop an item
*{{Adv menu icon|g}} - Grab/gather (pick up) an item
*{{Adv menu icon|p}} - Put an item into a container; or onto a pack animal
*{{Adv menu icon|r}} - Remove an item you are wearing or from a container
*{{Adv menu icon|w}} - Wear an item*
*{{Adv menu icon|I}} - Interact with an object in inventory (unstick a weapon, refill waterskin etc)
*{{Adv menu icon|x}} - Craft (butcher, create item)
*{{Adv menu icon|b}} - Found a site and build
*{{Adv menu icon|q}} - Sheathe your weapon
}}
<nowiki>*</nowiki>If you want to get a specific item from your backpack to your hand (wield weapon), {{k|p}}ut current item into the backpack, and {{k|r}}emove the other item from the backpack.

{{KReference|Information|
*{{Adv menu icon|z}} - Status
*{{k|Space}} - Advance/Clear Messages
*{{Menu icon|a}} - View Announcements
*{{Adv menu icon|Q}} - Adventure log (tasks, map, et cetera...)
}}

{{KReference|Food/Sleep|
*{{Adv menu icon|Z}} - Sleep
*{{Adv menu icon|e}} - Eat or drink something
}}

{{KReference|Travel Keys|
*{{k|8|2|4|6|7|9|1|3|↑|↓|←|→}} - Move.
*[[File:Adv ui descend.png]]{{k|<}} or [[File:Adv ui ascend.png]]{{k|>}} - Move into/out of tunnel (respectively).
*{{Adv menu icon|T|c|num=1}} - Start/stop cloud viewing.
*{{Adv menu icon|T|d|num=1}} - Visit location.
*{{k|h}} - Display/hide instructions
*[[File:Adv ui T.png]]{{k|m}} - Display/hide map.
*{{Adv menu icon|S}} - Sneak.
*{{Adv menu icon|T|K|num=1}} - Display all nearby tracks and strongest odor.
*{{Adv menu icon|Q}} - Adventure log/information.
*{{Adv menu icon|T|Z|num=1}} - Sleep/wait.
}}

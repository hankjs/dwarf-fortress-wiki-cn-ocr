每个[[dwarf|矮人]]最好的朋友,而且确实是要塞中的摇滚巨星!'''酿酒师'''将各种[[crop|植物]]和[[honey|蜂蜜]]酿造成[[alcohol|酒精]]饮料,如麦芽酒、啤酒和蜂蜜酒。

酿酒师需要一个[[still|蒸馏器]]、一个可酿造的[[crop|植物]](或一罐[[honey|蜂蜜]]),以及每个工作一个空的[[barrel|桶]]或防水[[pot|罐]]来酿造饮料。

无论[[skill|技能]]如何,酿酒师每单位可酿造植物生产五单位酒精,以及该植物的[[seed|种子]]。由于作物[[stack|堆叠]],这使得酿酒师可以与高天赋的[[herbalist|草药师]]和[[grower|种植者]](他们更有可能生产更大的植物堆叠)很好地协同,并可能快速导致蒸馏器[[clutter|杂乱]]。酿酒师会将"[[Plump helmet|肥大头盔菇]]"变成"[[Dwarven wine|矮人酒]] [5]",并将"肥大头盔菇 [5]"变成"矮人酒 [25]"。两个动作都填满一个空桶(尽管数量不同),并且大概需要相同的时间来酿造。

与许多农业劳动不同,没有桶或罐就无法进行酿造。如果你的要塞有比桶更多的食物,这可能会成为一个问题,因为任何多余的桶都会立即被食物填满。限制你的食物[[stockpile|储存区]]可以存储的桶数量可以帮助保持一些桶空着用于酿造。

酿造植物总是产生种子,所以你不必担心下个季节无法重新种植你的作物。

== 技能、质量和想法 ==
对特定类型酒有[[preference|偏好]]的矮人在饮用它时会有快乐的想法。无法满足偏好的矮人永远不会从饮酒中获得快乐的想法。酒没有[[quality|质量]]等级,但饮酒的各种快乐[[thoughts|想法]](最近喝了一杯精致的/相当不错的/美妙的/真正颓废的/传奇的饮料)来自整堆酒精的价值{{cite forum|120870.msg3901346#msg3901346}}。还有人推测喝家酿酒比喝同类型的进口酒更受欢迎,影响想法。{{verify}}

矮人喜欢一些多样性 - 他们会因反复消费同样的老酒而产生不快乐的想法(然而,这永远不会发生在他们最喜欢的饮料上。)幸运的是,随着水果采集的增加,保持广泛的饮料选择相当容易。

== 故障排除 ==

如果你在酿造饮料时遇到困难,检查你的蒸馏器。如果"酿造饮料"选项是:

* 完全缺失

: 你的raws已损坏,可能是因为安装了过时的mod(包括图像集)。在一个干净的文件夹中重新安装游戏,并只使用为你的版本更新的mod。


* 红色文本

: 游戏认为你没有酿造所需的适当资源。检查:

# 你有被认为可酿造的材料。输入z(状态),"厨房"标签来检查这一点。
# 你有空闲的桶或上釉的罐。如果不确定,再制作一些。
# 如果蒸馏器链接到一个或多个[[stockpile|储存区]],确保它们包含所有需要的组件(包括桶)。添加更多链接,移除当前链接,或者只是重建蒸馏器。
# 如果蒸馏器在启用了车间限制的[[burrow|地洞]]内,那么可酿造植物也必须在地洞内。
# 你的要塞的可酿造物品没有全部被搬运工作阻塞。带有活动"存储物品"工作的容器中的物品无法用于任何其他用途,包括酿造。{{bug|9004}}
# 启程购买的[[fruit|水果]]装在[[bag|袋子]]里,因此无法直接酿造{{bug|7423}}。要酿造它们,你必须首先[[Garbage dump|倾倒]]它们然后回收它们。


* 白色文本

: 游戏认为你应该能够酿造。如果你的矮人没有酿造,检查:

# 你至少有一个启用了"酿造"劳动的空闲矮人。
# 你的酿酒师可以到达蒸馏器。
# 如果你的任何酿酒师被分配到地洞,地洞需要包括蒸馏器和可酿造植物。{{bug|2262}}
# 你的要塞的可酿造物品没有全部被搬运工作阻塞。带有活动"存储物品"工作的容器中的物品无法用于任何其他用途,包括酿造。{{bug|9004}}

'''World tokens''' are used to define the [[Advanced_world_generation#Parameter_sets|parameter sets]] used in creating worlds, which are stored in <tt>prefs/world_gen.txt</tt>, though the file might not exist until you save a parameter set. You can change all of these in-game in the world generation menu by entering ''Detailed mode'', or by editing the plain-text file directly. For information on generating worlds with particular geographic features, see [[Advanced world generation]].

World parameters can be read by the [[Lua scripting|Lua runtime]], stored in ``world.param[lowercase token name]``.

=World Token Index=

Select the token you want documentation on and you will be taken to the proper section of the Advanced world generation documentation. Note that the tokens are listed in the same order that the game saves (saved) them in.

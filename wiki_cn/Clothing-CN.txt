'''衣物'''是由[[cloth|布料]]、[[silk|丝绸]]或[[leather|皮革]]制成的物品，由有智慧的生物穿着和拥有。特定衣物的可用性因文明而异；每个文明都有自己可以生产的衣物套装。例如，在要塞模式中，凉鞋和鞋子属于同一衣物类别，但矮人只能生产后者，而凉鞋必须从死去或被俘虏的敌人身上剥下来。矮人的衣物不分性别；男性矮人很可能会穿上裙子，女性当然也可以穿裤子。

在游戏引擎中，衣物和[[armor|护甲]]之间没有根本区别，普通衣物偶尔能够阻挡攻击这一点强调了这一点。衣物会随着时间磨损。一些可穿戴的皮革物品，包括皮甲、头盔和靴子确实是护甲，不会被平民穿着。护甲可以被视为金属衣物，更厚且有更好的阻挡攻击的机会。但是，护甲不会经受标准磨损，也不会被平民自动装备。有关特定衣物及其材料的详细信息，请参见：[[Armor#Material_requirements|护甲#材料要求]]。

尽管许多生物，包括非人形生物，可以穿各种衣物，但只有矮人、[[human|人类]]、[[elf|精灵]]、[[kobold|狗头人]]和[[goblin|地精]]会在视觉上显示穿着不同的衣物。这是因为这些种族都有专用的精灵图集。虽然其他生物可以穿衣服，但不会出现在它们的精灵图上，但点击它们并检查其库存可以显示它们穿着某些东西，即使在现实中这没有意义。

从图形上看，某人衣物的精灵图会与其职业相似，这可以在选项中更改，使衣物始终由染料表示。选择生物并查看其财产时，衣物会以其真实颜色显示，如果生物有显示其所穿衣物的肖像，也可以很容易地看到，以及它们的质量。在[[object testing arena|物品测试竞技场]]中，生成生物时，精灵图上的任何可见衣物都将完全随机化，以帮助区分个体。但是，竞技场中生成个体的肖像和精灵图标中的衣物颜色将具有其默认的通用颜色。

== 用途 ==
[[File:dwarf_clothes_preview.png|right]]平民穿衣服的主要原因是为了端庄，详情请参见下面的[[Clothing#Thoughts|想法]]部分。一些矮人还会[[need|需要]]奢华，因此具有[[quality|质量]]或[[decoration|装饰]]加成的衣物会满足这一需求。鞋类的一个特殊好处是防止你的矮人赤脚踩到可能危险的[[contaminant|污染物]]。

由于可以应用于成品衣物的多个[[value|价值]]修正器，它们可以作为有用的[[trade|贸易]]商品——即使是破烂的衣物也能卖出不错的价格，所以这样的衣服可以使用两次。

与护甲相比，衣物对攻击的保护要少得多，尤其是来自金属武器的攻击，但皮革衣物可以提供对较小动物攻击的显著保护。目前，衣物无法提供对寒冷环境的保护。如前所述，某些个体（如[[Adventure mode|冒险模式]]中创建的角色）的上身衣物的质量可以在肖像中看到。在开始该模式的游戏之前，玩家可以为自己选择任何类型的可穿戴物品，以点数为代价。在这些角色肖像中，可以很容易地看到衣物的质量，因为衣物的质量越高，装饰就越华丽。这可以在下图中看到：

[[File:coat_quality_preview.png|thumb|708px|center|从左到右，人类外套的质量等级分别为标准、-精制-、+精细制作+、*优质*、=杰出=、☼大师级☼和神器。]]

在冒险模式中创建角色时，玩家可以通过点击[[File:quality_increase_button.png|16px]]和[[File:quality_decrease_button.png|16px]]来选择某些衣物和武器的质量，更高的质量等级花费更多点数。请注意，某些衣物不会为每个质量等级显示独特的图形，并且玩家永远无法从一开始就选择神器质量。

== 制作 ==
衣物可以在[[clothier's shop|裁缝铺]]用布料制作，或在[[leather works|皮革作坊]]用皮革制作。它将按照执行工作的生物的[[size|尺寸]]制作，详情请参见下一节的尺寸部分。染料不能直接涂在衣物上——要制作染色的衣服，必须先染制作衣服的布料或制作布料的线。通常成对穿着的衣物，如手套、鞋子和袜子，将成对制作，尽管创建的两件物品的质量等级可能不同。目前，任何衣物工作都会消耗一整单位的布料或皮革，无论生产的件数大小如何。

要最大化价值，使用自定义储存区收集高质量的染色布料并将其链接到生产裙子或长袍的裁缝工坊，并将工坊分配给熟练工人。

请记住，在[[leather works|皮革作坊]]生产的某些物品构成皮革[[armor|护甲]]而不是衣物。这包括"皮甲"（指皮革上身护甲）、皮革护腿、皮靴（高筒和低筒）和皮革头盔。这种区别很重要，因为这些物品不会被平民自动拾取和装备，不会磨损（但当然可能在战斗中遭受磨损），并且必须存储在[[armor|护甲]]储存区而不是如下所述的[[finished goods|成品]]储存区中。

== 尺寸 ==
衣物和护甲都有与之相关的尺寸，为一种尺寸的生物制作的装备不能被足够大或小的生物穿着——可接受的范围在生物尺寸的±1/7之间。<sup>[http://www.bay12forums.com/smf/index.php?topic=140544.msg6843526#msg6843526][http://www.bay12forums.com/smf/index.php?topic=169696.msg8284471#msg8284471]</sup>对于矮人来说，这适用于人类和[[troll|巨魔]]穿着的衣物和护甲（会显示为''大号''<sup>*</sup>）以及狗头人（会显示为''小号''）；地精和[[elf|精灵]]的尺寸与矮人相同，因此它们的衣物和护甲可以装备，而不仅限于[[Melt item|熔化]]（对于金属护甲）或用作[[trade|贸易]]商品（特别是一旦[[Decoration|装饰]]后）。

:''(*) 请注意由于术语原因的一个例外：由[[large rat|大老鼠]]皮革制成的物品将显示为"大老鼠皮革______"。由大老鼠皮革制成的人类尺寸手套将是"大号大老鼠皮革手套"。（没有提供皮革的非大号"老鼠"，所以虽然令人困惑，但这是明确的。）''

任何不是''大号''或''小号''的衣物/护甲都是均码，可以被任何矮人穿着，从最小的孩子到最大的成人。

你的矮人可以制作任何尺寸的衣物和护甲，以满足非矮人要塞居民的需求。为此，请从各自的工坊请求制作衣物或护甲，然后点击工坊或工作订单菜单中任务的[[File:Mag glass icon.png]]以指定尺寸。或者，由生物制作的衣物将自动为该生物调整尺寸。为美洲狮人或鬣狗人调整尺寸的衣物允许人类和矮人尺寸的种族都穿着。

== 储存 ==
衣物是[[finished goods|成品]]，如果未被矮人认领，将存储在成品[[stockpile|储存区]]中，可选择放在[[bin|箱子]]中。衣物可以根据其在身体上的穿着位置在储存区内进行过滤，并分类为护甲（指上身衣物，而不是必须存储在[[armor|护甲]][[stockpile|储存区]]中的防护[[armor|护甲]]）、手套、鞋类、头饰或腿饰。

矮人会将他们的个人衣物存储在他们的[[zone|房间]]中，直接放在地板上或放在[[cabinet|柜子]]中。[[personality facet#ORDERLINESS|整洁]]（又称有条理）的矮人很少会放弃他们的破烂衣物，而是在他们的房间里积累大量磨损的衣服。拥有的衣物不能被其他矮人认领或搬运，但如果当前没有穿着或不在主人的住处，它最终会在一个季节后恢复为无主状态，使其可用。

请注意，存储在划分为矮人卧室一部分的[[cabinet|柜子]]中的拥有衣物在当前版本中永远不会恢复为无主状态，这可能导致大量破烂衣物逐渐在你的要塞[[bedroom|卧室]]中积累。有关更多信息，请参见下面的错误部分。

== 磨损 ==
[[File:wearing_clothes_example.png|thumb|300px|right|一个矮人孩子的衣物...最近接触过酒精。]]衣物随着时间的推移会[[wear|磨损]]；在穿着时，每两年会增加一级磨损。在正确存储时，无论是在（非垃圾）储存区还是在矮人的个人住处，它会非常缓慢地退化（每世纪一级磨损，如果使用[[DFHack]]则自动减少到每20年一级磨损）。磨损的衣物可能会在高人口要塞中迅速开始积累，并且可以被交易掉或销毁。请注意，通过[[atom smasher|原子粉碎机]]或岩浆销毁的[[Item_quality#Quality_grades|大师级]]衣物可能会给其创造者带来不愉快的想法，即使该物品已磨损。

请注意，如果穿着者受到伤害，特别是如果攻击穿透或来自更硬的材料，衣物也可能受损——即使是由[[Adamantine|特殊]]布料制成的衣物也是如此。

如果要塞中有合适的衣物，矮人会尝试自己更换磨损的衣物。（[http://www.bay12forums.com/smf/index.php?topic=124350.msg4133470#msg4133470 支持的bay12论坛帖子]）

===== 管理衣物 =====
生产足够的新衣物以满足大型繁忙要塞矮人需求是一项持续的工作，因为衣物会在不同时间磨损。即使在建立了足够的[[textile industry|纺织业]]或为原材料进行[[trading|贸易]]之后，矮人也会不断磨损他们当前装备的物品，需要制造新衣服。

除了上述策略之外，组织生产和减少流浪、废弃衣物普遍性的另一种方法是创建两个[[finished goods|成品]][[stockpiles|储存区]]。确保两个储存区都允许"护甲"、"鞋类"、"手套"、"头饰"和"腿饰"类别以接受衣物，并允许相关材料和质量。

第一个储存区仅用于新衣物，只会填充你创建的新衣物。它关闭了"从任何地方获取物品"开关，并被分配为仅从[[clothier's shop|裁缝铺]]工坊获取物品。这两条规则都可以通过相应储存区设置菜单右上角的图标进行调整。如果你分配熟练裁缝和足够的生产，这个储存区将迅速开始积累可供矮人穿着（或交易掉）的高质量衣物。

第二个储存区用于旧衣物，将收集磨损的、无主的衣物以及来自[[clothier's shop|裁缝铺]]以外来源的衣物。为此，它打开了"从任何地方获取物品"开关，这样矮人就会将要塞周围的任何无主衣物搬到这里。许多磨损的物品会在这里积累。

现在你可以在你的[[trade depot|贸易站]]附近创建具有类似规则的第三个储存区。这个储存区可以关闭"从任何地方获取物品"开关，并被分配为仅从你的旧衣物储存区获取物品。现在矮人会将所有磨损的衣物（可能还有一些从其他地方收集的较少磨损的物品）搬到非常靠近贸易站的箱子中的储存区，使它们易于识别和交易（因为与你要塞中所有其他箱子相比，到贸易站的距离非常低）。这样做的好处是可以处理磨损的[[Item_quality#Quality_grades|大师级]]质量物品，可以自由交易而不会给其创造者带来负面想法。

== 想法 ==
如果成年矮人裸体或穿着破烂的衣服，他们会产生不愉快的想法。婴儿不穿衣服，但所有成年和儿童矮人在有衣物可用时会自动认领并穿着衣物。矮人可能会因以下原因产生几种不同的与衣物相关的负面想法：
* 完全裸体
* 缺少上身覆盖物（衬衫、背心、裙子、长袍、斗篷或外套）
* 缺少下身覆盖物（裤子或裙子）
* 缺少鞋类（鞋子、凉鞋或袜子）
* 穿着破烂的衣物（具有"X"或"XX"磨损的物品，但不是小写"x"磨损）

衣物想法相当强烈，并且相互叠加。除非你生产足够的鞋类、上身和下身覆盖物品，否则很可能会发生[[Tantrum spiral|暴怒螺旋]]。幸运的是，分配给相关身体部位的[[armor|护甲]]也可以防止负面想法，并且具有不受磨损影响的优势，除了提供更好的保护之外。如果你告诉你的民兵矮人用他们的制服替换普通衣物，请确保在征召<s>炮灰</s>传奇[[cheese maker|奶酪制造者]]时有足够的护甲，因为即使你没有为他们准备制服，他们也会很乐意脱光衣服。你也可以在混合中包括一层布料"护甲"、护腿，尤其是袜子，以防止在真正护甲短缺的情况下出现裸体不快乐，但要警告：你的矮人不会自动更换磨损的制服分配的衣物，导致你的装备衣物的民兵矮人产生不同的不愉快想法。

如果你发现生产大量[[armor|护甲]]而不是衣物更容易，你也可以将所有平民矮人分组到小队中并为他们分配覆盖基本部位的制服。这不仅确保你不必为他们提供替换衣物，而且还确保你的整个人口至少受到稍微更好的保护。让每个人都分组到小队中，当你的士兵在其他地方忙碌时，如果有什么讨厌的东西出现在你的平民中间，也可能会派上用场。这样做的缺点是更重的护甲可能会减慢平民矮人的速度。

== 错误 ==
* 矮人在决定拾取替换品时会在他们碰巧所在的地方丢下磨损的衣物。这些丢弃的衣物可能会干扰储存区、工坊、建造订单等。{{bug|6048}}，并且是拥有的物品，因此不能轻易移动。{{bug|4403}}矮人稍后会回来收集衣物，这在[[trap design|陷阱]]、战场和[[minecart|矿车]]轨道周围可能特别危险。
* 矮人在他们的房间里积累了大量丢弃的衣物，填满了所有[[cabinet|柜子]]并覆盖了每个可用的瓷砖。{{bug|7680}}禁止这些物品将在几周后清除其所有权。{{cite forum|173188/7926370}}[[Utility:DFHack|DFHack]]提供了"cleanowned"命令来帮助清理混乱。
* 军事制服发放的磨损衣物不会被替换。{{bug|6039}}

== 袜子痴迷 ==
{{D for Dwarf}}
[[Image:deadly sock.png|thumb|right|牛头怪也喜欢袜子。]]
这是矮人对袜子痴迷的开始。很久以前，袜子在全国各地的矮人王国中很丰富，长胡子的人很少注意他们脚上穿的是什么。但在自豪的城堡诺格里索格因袜子短缺的争端而在一场可怕的内战中自我毁灭之后，他们对袜子的生产完全一无所知，宇宙中各地的整个矮人种族发誓永远不会让这种情况再次发生。矮人确保储备羊毛、丝绸甚至布料鞋类，以防不可想象的事情再次发生。孩子们在停止吮吸母亲的乳房之前就被伟大袜子战争的故事吓坏了，以灌输他们正确的袜子囤积行为，并让调皮的孩子表现："如果你不停止拉卡滕的辫子，西布雷克，所有的袜子都会消失！"

矮人对袜子短缺的恐惧如此之大，以至于他们会牺牲自己的福祉甚至生命，以确保不会忽视一只袜子。不知道各地的矮人怎么可能知道一个遥远且完全不重要的大陆上被遗忘的次要世界上的一个矮人文明是如何自我毁灭的。大多数人认为这再次是阿莫克在他们潮湿、不断中毒的大脑上的触摸。

矮人和他的袜子之间的纽带既美妙又可怕。最顽强的战士发现他的袜子如此温暖舒适，以至于他经常被看到只穿一只靴子——他为什么需要钢来保护他的另一只脚？他已经在上面穿了袜子！在失去未穿护甲的脚于地精的刀刃后，他会在知道至少他的脚上还穿着袜子的知识中安慰自己。

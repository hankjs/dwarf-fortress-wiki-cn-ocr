'''矮人'''(单数,'''Dwarf''')是[[Stupid dwarf trick|"智能"]]、[[alcohol|酒精]][[Creature_token#ALCOHOL_DEPENDENT|依赖]]的{{Catlink|Humanoids|类人}}[[creature|生物]],是[[fortress mode|要塞模式]]的特色[[civilization|种族]],也可以在[[adventurer mode|冒险者模式]]中游玩。它们以健壮的体格和突出的[[beard|胡须]](在雄性身上)而闻名,胡须从出生开始生长;矮人比普通[[human|人类]]更强壮、更矮、更结实、体毛更多,具有增强的周围环境感知能力,并拥有完美的黑暗视觉。矮人既生活在从[[mountain|山腰]]雕刻的精致地下[[fortress|要塞]]中,也生活在地上[[hillock|小丘]]上,是天生的[[miner|矿工]]、[[metalsmith|金属匠]]和[[stone crafter|石匠]],最重视获取[[wealth|财富]]和[[value|稀有]][[metal|金属]]。

矮人文明通常与[[human|人类]]和[[elf|精灵]]形成(大多数)和平的、基于贸易的关系,但与[[goblin|哥布林]]是死敌,并将[[kobold|狗头人]]视为小烦恼。矮人婴儿在出生一年后成为[[children|儿童]],在十八岁生日时成长为成年人,寿命约为150-170岁。

训练有素的矮人在战斗中是威胁;它们是唯一能在被多个敌人围攻时进入[[martial trance|武术恍惚]]的种族,授予它们重大战斗加成,它们对采矿和金属加工的重视确保了对最好[[weapon|武器]]和[[armor|盔甲]]的访问。然而,它们无法骑乘[[mount|坐骑]],总是徒步战斗。但请注意,这仅适用于要塞模式中玩家控制的矮人,因为如果它们是[[siege|围攻]]的入侵力量,它们可以使用坐骑。

矮人说[[Main:Dwarven language|矮人语]]。

一些非矮人[[Preferences|喜欢]]矮人,因为它们的''胡须''。

== 要塞模式 ==
矮人是要塞模式中当前的默认种族,也就是说,唯一可以在不进行[[modding|模组]]的情况下游玩的种族。作为[[trading|贸易]]种族,矮人每年[[Calendar|秋季]]会派遣[[caravan|商队]]。这些商人会带回有关要塞财富和货物的故事,这将吸引[[immigration|移民]],[[outpost liaison|前哨联络官]]将带来有关世界动态的消息,主要集中在玩家的文明上,包括[[mayor|市长]]选举结果、各种难民群体在[[war|战争]]期间从其要塞和小丘逃离,以及[[monarch|君主]]更替,如果前任死亡有或没有继承人。

矮人偶尔会[[Strange mood|受到启发]]创造[[legendary artifact|传奇神器]],一件工艺精湛且价值巨大的物品。在这个任务中成功的矮人,在大多数情况下,将成为该职业的[[skill|传奇]]工人;但是,如果没有适当的材料,矮人将会[[insanity|发疯]]。梦想创造伟大艺术作品的矮人渴望奇异心境降临到他们身上。

作为地下生物,矮人不喜欢太阳,当在[[Cave_adaptation|在黑暗中待了足够长的时间后]]暴露于太阳下时,它们会经历负面思想和呕吐,这会更糟,持续时间更长,它们在地下待的时间越长。

矮人根据其[[personality facet|性格特征]]对压力做出反应;有些会以恐惧或暴力反应,除非它们已经变得足够[[discipline|有纪律]](或麻木)对待这些事情,而其他人除非压力特别个人化,否则不会感到任何东西。当被不幸事件推向不快乐时,矮人会根据其性格进入[[Mental breakdown|情绪崩溃]],从[[depression|抑郁]]到攻击性[[tantrum|暴怒]] - 发现暴怒的矮人掀翻家具、伤害他人并普遍闹事并不罕见。尽管如此,它们也有强烈的[[justice|正义]]感,那些破坏财产或其他矮人的人可能会发现自己被监禁,或者 - 在极端情况下 - 在[[hammerer|锤击者]]的体罚接收端,尽管来自要塞守卫的无意致命殴打并不罕见。虽然游戏的旧版本以其"暴怒螺旋"而闻名,矮人会在连锁反应中相互激怒并在过程中摧毁要塞,但这些在当今并不常见,尽管压力大的矮人仍然是丰富的[[fun|乐趣]]来源。

== 需求 ==

作为活生生的生物,矮人需要[[Thirst|饮水]]、吃[[food|食物]]和[[sleep|睡觉]]才能生存。平均而言,矮人每18天需要喝一次,每35天吃一次,每40天睡一次。矮人更喜欢[[alcohol|酒精]],并且独特地依赖酒精;如果剥夺足够长的时间,矮人会遭受戒断症状,工作速度会变慢。

矮人还有各种情感[[needs|需求]],这些需求源于其[[personality facet|性格特征]]和[[Personality value|价值观]]。典型的矮人可能需要获取[[Finished goods|物品]],吃[[Item value|好]][[Prepared meal|餐]],维持其[[relationship|关系]],磨练其[[skills|技能]](通常在[[Industry|制作]]中,但也可能是[[Combat skill|武术]]或[[Library|智力]]),或享受[[performance|表演]]。

== 冒险者模式 ==
矮人要塞可以在[[Adventurer mode|冒险者模式]]中探索,包括其世界生成和玩家制作的形式。它们可以是矮人大小[[armor|盔甲]]的良好来源,因为[[human|人类]]盔甲对矮人来说太大了。由于地下[[Fortress|要塞]]和[[mountain halls|山区大厅]]的布局被认为[http://www.bay12forums.com/smf/index.php?topic=169696.msg8565801#msg8565801 太混乱]而无法导航,{{v|51.01}}冒险者无法从那里开始。矮人冒险者必须从其他[[site|地点]]类型开始,如[[hillocks|小丘]]或由另一个[[civilization|文明]]建立的地点。

矮人具有在黑暗中完美看清的显著能力,无论是在夜间还是[[underground|地下]]。它们也可以在与要塞模式中相同的条件下进入[[martial trances|武术恍惚]]。由于它们的体型,大多数人类武器必须由矮人双手挥舞,当然,除了矮人也可以制作的武器。

== 伦理 ==
[[File:Dancing dwarves crayon.jpg|thumb|250px|这是一幅蜡笔画。所有矮人工艺都是最高质量的。物品上是蜡笔绘制的矮人图像。矮人在笑。矮人在跳舞。物品上有[[Main:ThreeToe|Zach]]和[[Main:Toady One|Tarn Adams]]的蜡笔签名。]]

在道德上,矮人[[ethics|伦理]]最接近人类和精灵伦理,与狗头人和[[animal people|动物人]]伦理有些一致,与哥布林伦理强烈不一致。与精灵不同,矮人认为吞食死敌不可想象,不会屠宰或消费智能生物(哥布林认为这是个人问题)。它们完全反对出于任何理由的任何形式的酷刑,不像精灵、人类、狗头人和动物人(他们认为某些形式的酷刑是可以接受的),尤其是哥布林,他们认为所有酷刑都是可以接受的。矮人容忍动物战利品,但避开保留有智生物战利品的人,并认为保留其他矮人战利品的人令人震惊。矮人认为杀死动物、敌人和植物完全可以接受,不像精灵、狗头人和动物人。这方面的一个例外是杀死中立生物,只要杀戮已被正式命令,这是允许的。被发现参与攻击、盗窃、非法侵入或破坏公物的矮人将受到严厉惩罚;某些罪行,如杀死其他矮人、违背誓言、奴役和叛国,可处以死刑。另一方面,撒谎被视为个人问题。

矮人高度重视工艺,并深深尊重法律、忠诚、家庭、友谊、真理、艺术作品、技能和努力工作。它们对武术能力、合作、公平、独立、坚忍、商业、欢乐、休闲时间和毅力给予一定程度的重视。它们对自然不尊重。

== 社区观点 ==
矮人在玩家中积累了迟钝的声誉,尽管公平地说,这更多是游戏本身(及其某些后果)的功能,而不是矮人的错。矮人愚蠢的例子数不胜数;例如包括[[dodging|闪避]]到稀薄空气中(从悬崖和进入[[river|河流]]),从不考虑[[water|水]][[flow|流]](被冲下[[waterfall|瀑布]]到几个[[z-level|z层]]下面的水墓),总是走简单的[[path|路径]](即使通过被有毒[[forgotten beast|遗忘野兽]]血液[[syndrome|毒害]]的池塘),在树枝上爬行并掉进树枝悬挂的湖泊或池塘,从错误的一侧建造[[construction|建筑]](将自己困在里面),[[channel|挖掘]]自己<s>正</s>曾站立的地板,徘徊去做危险的事情(当[[giant cave spider|巨型洞穴蜘蛛]]明显潜伏时[[web|收集蜘蛛网]]),以及普遍无视危险情况(即使使哥布林将它们变成水平的哥布林仅有两英尺远,也要搬回友好的[[corpse|尸体]])。

矮人也有极易激动的声誉,在最轻微的挑衅下就会感到压力。这个问题因游戏先前版本中矮人的敏感性而加剧;它们很容易变得沮丧,为看似微小的原因(例如暂时缺乏[[wear|未穿]]的[[clothing#The_great_sock_obsession|袜子]])而发脾气,失控。这不可避免地导致玩家强调"矮人风格":故障安全设计、用机械代替矮人力量、用[[magma|岩浆]]玩乐、抓住对环境的控制、杀死所有可爱的毛茸茸动物、将整个地方采空、等等。被称为[[elf|精灵]]是一种巨大的侮辱,暗示玩家容易坐在那里凝视树木,生活在世界中而不是将其弯曲到他们的意志。

在功绩或[[fun|乐趣]]的描述中,通用名称''[[main:Urist|Urist]]''经常用于代替任何特定矮人[[name|名字]],通常是因为默认矮人名字复杂、随机且难以记住。矮人的其他几个昵称也存在;有些不如其他人礼貌。

== 神话 ==
在现实神话中,矮人很像人类,但通常更喜欢住在地下和/或山区。在他们的要塞中,他们积累了[[gold|金]]、[[silver|银]]和[[gem|宝石]]的财富,并花时间制作昂贵的武器和盔甲。它们是著名的矿工和铁匠,尽管像人类一样,它们可以专门从事任意数量的行业。通常比人类矮,它们平均更结实、体毛更多,通常留着浓密的胡须。虽然跑步慢、骑术差,但矮人是优秀的战士和他们据点的捍卫者。矮人有锻造魔法物品的能力,这展示了他们文化和物种的天然工艺。例如,矮人铁匠创造了一些神话中最伟大和最强大的物品,这启发了游戏中的[[strange mood|奇异心境]]和[[legendary artifact|传奇神器]]。

== 关于矮人的神话 ==
{{D for Dwarf}}

矮人,就像圣经英雄参孙一样,实际上从他们的胡须中获得力量。(回想杰克逊的《指环王》中的金雳,"不是胡须")。在其他物种的哲学家和生物学家中,有一些关于酒精和矮人表现出的近乎神一般的特性之间是否存在联系的推测,大多数矮人对此的回应是一阵咆哮般的笑声,然后把冒犯生物的头砸进去,咯咯笑着说,"''有''联系吗?",同时突然决定由于意外的快乐提升而创造神器。

这让我们想到了为什么矮人看到同伴矮人杀死生物时会感到悲伤的问题。在这方面,大多数物种只是部分正确地假设矮人后悔生命的丧失。矮人感到不安、偶尔变得忧郁并经常发脾气的真正原因是,由于后悔'''他们'''不是该生命丧失的原因。当被杀生物属于以下类别之一时尤其如此,按后悔程度升序排列:
      A) "LnT"    大而有威胁性
      B) "SCnI"   小、可爱且无辜
      C) "FB"     遗忘野兽(泰坦、遗忘野兽和其他大型"乐趣"也包括在此)
      D) "g"      哥布林
      E) "k"      狗头人
      Z) "ë"      精灵
